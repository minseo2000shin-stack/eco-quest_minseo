# Windows React Hello World

윈도우 환경에서 바로 띄울 수 있는 Vite + React 헬로월드 예제입니다.

## 사전 준비
- Node.js 18.18+ (현재 로컬: v22.17.0)
- npm 9+ (현재 로컬: 10.9.2)
- Yarn도 사용 가능 (선택 사항)

## 설치
```bash
cd win-setup/hello-world
npm install        # 또는 yarn install
```

## 실행 (개발 서버)
```bash
npm run dev -- --host
```
- 브라우저에서 `http://localhost:5173` 접속

## 빌드 & 미리보기
```bash
npm run build
npm run preview
```
